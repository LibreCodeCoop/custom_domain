# Custom Domain

Use custom domains on a Nextcloud instance by mapping a company code or subdomain to:

- a Nextcloud group
- a trusted domain entry
- a company-specific theme asset set

See the [installation and administration guide](docs/README.md) for setup, customization, theming, CLI commands, tests, and runtime requirements.
